OLED Display Library - compatible with Arduino 1.0
*****************************************************************

Designed for use with SSD1306 controller driving a 128x64 resolution OLED display

Download to Arduino IDE 'libraries' folder. Restart of IDE required.

Created November 26th 2018 based on work by Sonal Pinto https://github.com/SonalPinto/Arduino_SSD1306_OLED
